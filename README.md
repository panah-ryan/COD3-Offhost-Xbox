# COD3-Offhost-Xbox
Offhost menu code for COD3 on xbox 360, contains code from other released aimbots in the community. Code is messy and 90% of the menu functions don't do anything.
Basic ESP and Drawing on Screen is there, you can also modify Cvars
Aimbot does not work properly, never could get it to aim properly probably a skill issue on my part.
